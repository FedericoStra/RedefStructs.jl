RedefStructs.jl

Documentation for RedefStructs.jl.

This package provides the macro @redef, which allows to create structures which are redefinable.

Example

julia> using RedefStructs

julia> @redef struct S
           s::String
       end

julia> S("Hello").s
"Hello"

julia> @redef mutable struct S
           n::Int
       end

julia> S(42).n
42
